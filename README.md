Projeto desenvolvido durante o curso de HTML5 e CSS3 do professor Gustavo Guanabara (Curso em Vídeo).
O objetivo foi criar um site informativo sobre o Google Glass, aplicando conceitos de estruturação semântica, estilização com CSS e organização de pastas.

O site é composto por múltiplas páginas — como apresentação do produto, especificações técnicas, galeria de fotos, multimídia e formulário de contato — demonstrando boas práticas de layout e navegação entre seções.

Tecnologias utilizadas:

HTML5

CSS3

Objetivo de aprendizado: consolidar fundamentos do desenvolvimento web, como marcação semântica, links internos, listas, tabelas, formulários e organização de arquivos em um projeto completo.
